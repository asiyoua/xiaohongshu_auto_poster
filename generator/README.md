# 小红书图片生成器 (bxz-xhs)

> 从内容到发布，全自动化小红书图文创作工具

## ✨ 功能

- 📝 **智能分析**：自动分析内容，生成三种差异化大纲策略
- 🎨 **多样风格**：10 种视觉风格 × 8 种布局，自由组合
- 🖼️ **AI 生成**：使用 Gemini AI 生成高质量图片
- 📱 **自动发布**：一键发布到小红书（扫码登录，长期有效）

## 🚀 快速开始

```bash
# 1. 生成小红书图片
/bxz-xhs
[粘贴你的内容]

# 2. 发布到小红书
/bxz-xhs-publisher ~/Myxhs/{topic-slug}
```
